# Changelog

All notable changes to CourseFrame are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] — 2025

### Added
- Initial public release
- AI generation mode supporting Claude, GPT-4o, Gemini, and Copilot
- Manual block-by-block building mode (no API key required)
- 11 block types: Text, Callout, Image, Accordion, Tabs, Quiz, Process, Timeline, Sorting, Button, Divider
- Full Rise 360-style preview with sidebar navigation and progress bar
- Inline editing — click any text in the preview to edit
- Brand colour customisation with 8 presets + custom hex + colour picker
- Pantone Mocha Mousse (2025 Colour of the Year) as default accent
- Desktop / Tablet / Mobile preview toggle
- Single self-contained HTML file export (~75–90kb, works offline)
- Block controls: move up/down, duplicate, delete
- Lesson management: add, rename, reorder lessons
- API key stored in-browser only — never transmitted to any third party
- MIT licence — free forever
